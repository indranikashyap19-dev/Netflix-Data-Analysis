# Netflix Data Analysis

This project explores and analyzes Netflix movies and TV shows dataset using Python.

## Objectives
- Understand content distribution (Movies vs TV Shows)
- Identify most popular genre
- Analyze content growth over years  

## Key Insights
- Most content on Netflix consists of Movies
- Drama is the most common genre
- Content has increased rapidly after 2015

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project File
- Netflix_Data_Analysis.ipynb
